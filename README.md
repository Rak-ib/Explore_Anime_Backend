# Anime Catalog Backend

## Overview
The Anime Catalog Backend is built with **NestJS** and **MongoDB**. It serves as the backend for a platform that allows users to explore anime based on the anime's name and category. Only the admin has the privileges to add, update, or delete anime entries, ensuring control over the anime database. In the future, the platform will include user authentication, notifications, and a better overall user experience.

## Features
- **Explore Anime**: Users can search for anime by name or category.
- **Admin Control**: Only admins can add, update, or delete anime details.
- **Future Enhancements**: The project will soon include login and registration features for users, along with notification capabilities.

## Tech Stack
- **NestJS**: A framework for building efficient, scalable Node.js server-side applications.
- **MongoDB**: A NoSQL database used to store anime data.
- **Axios**: A promise-based HTTP client to make API requests from the frontend (optional, for future reference).
